# 1. 프로젝트 설명
리엑트와 일렉트론을 활용한 데스크탑용 todo앱 만들어보기

# 2. 실행 방법
## yarn start
- 해당 명령어는 리엑트를 실행시켜주는 명령어이다.

## yarn build
- 해당 명령어는 OS에 맞는 데스크탑용 앱을 만들어줍니다.\
